# BetterTouchTool (BTT) - scruth-config
BTT ([folivora.ai](https://folivora.ai/)) is software for managing customizations to Mac mouse and keyboard behavior.
It is very deep, customizable and complicated.

We have a lifetime license ([BetterTouchTool](https://start.1password.com/open/i?a=VED5EEGDNFCSXPODWNIFMVTYA4&v=kmhwtn4fzf2vhyq3luti556edu&i=gi4wzntikgptfr7wdn3flhshdi&h=my.1password.com) in 1password).

# Scruth BTT presets
I manually exported [scruth-config.bttpresetcompressed](./scruth-config.bttpresetcompressed) from the BTT UI.

Scruth customizations were assigned (via BTT UI) to the "scruth-config" Presets and then that was exported.

Presumably, manually importing that preset is trivial on a new system.
